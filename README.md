# kahlua

An effort to make the Lua interpreter readable.

Also fixes Lua by adding 0-based indexing.

# Installation

(Works with [Emeralds](https://github.com/Oblivious-Oblivious/Emeralds))

```sh
em build app release
```

## Usage

Run the interpreter

```sh
./kahlua
```

Interpret a file

```sh
./kahlua <file>
```
